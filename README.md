# Usage

Репозиторий содержит подмодули (submodules). Для клонирования с подмодулями:

    git clone --recurse-submodules https://github.com/donaldbaiden/tpmp-lab2.git

# Modules

## tpmp-lab2-task3 (Структура MARSHRUT1)

Сборка:

    cd tpmp-lab2-task3
    make
    ./marshrut

## tpmp-lab2-task4 (Школьник — файловая обработка)
